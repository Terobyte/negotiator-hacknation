The problem

You negotiate twice a decade.
They negotiate fifty times a day.

  • Same two-bedroom, same route: one company says $2,000, another says $6,400
  • The cheap quote isn’t real — it grows on delivery day, when your stuff is already on their truck
  • “The truck is already in your area.” It isn’t. “This price expires today.” It doesn’t.
  • You call one company at a time. They know you won’t call back.

The part that matters

Honesty is a type, not a prompt

# negotiator/core/contracts/models.py — real code
_GATE_CAPABILITY = object()

class ApprovedUtterance(Contract):
    """A gate-issued capability, not a data container."""
    def __init__(self, **data):
        if data.pop("_gate_capability", None) is not _GATE_CAPABILITY:
            raise TypeError("ApprovedUtterance can only be issued by the honesty gate")
  • The speech engine takes exactly one input type — and only the gate can construct it. There is no second path to the speaker
  • Invented number → BLOCK · your budget about to slip out → BLOCK · a block is never dead air — it stalls like a human and redraws

The hot path

Ready for a real phone line

  • Barge-in arbiter: the human always wins the mic — the agent cuts itself off mid-word, deterministically. It can also hold a tactical pause: silence as a negotiation move
  • Latency is measured, not promised: per-hop mouth-to-ear report straight from the call journal — ≤ 800 ms simulated, ≈ 1.1 s over Twilio PSTN. No sub-500 ms fairy tales
  • Three transports, one interface: Twilio PSTN · WebRTC · ElevenLabs WS, behind a degradation ladder — live → sim → TTS cache → recording. The demo cannot die
