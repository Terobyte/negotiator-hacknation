# Counter-agent exports

These files are vendor-neutral, reviewable source configurations for the three ElevenLabs Agents
Platform personas. Phase 1 deliberately does not create or mutate agents in ElevenLabs. Import each
JSON in the dashboard (or translate it with the future provisioning adapter), choose a voice, and
attach a phone number when live credentials are available.

Canonical call order: `lowball_broker` → `rushed_dispatcher` → `pressure_closer`.
